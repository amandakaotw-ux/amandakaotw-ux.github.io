快速修正版使用方式：

1. 解壓縮這個 ZIP。
2. 把裡面的三個檔案放到 GitHub 最外層，取代原本三個檔案：
   - index.html
   - style.css
   - script.js

3. images 資料夾不要動。這版會讀你現在 images 裡的：
   - 01_hero_complete.jpg
   - 02_card_onigiri.jpg
   - 03_card_latte.jpg
   - 04_card_oden.jpg
   - 05_card_potato.jpg
   - 06_activity.jpg
   - 07_form_donation_cta.jpg

4. 回 GitHub Desktop：
   Summary: quick fix final website
   Commit to main
   Push origin

5. 等 1–3 分鐘，網站 Ctrl + F5。
