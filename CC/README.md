# Cinderella × Uber AI

可直接部署至 GitHub Pages 的完整靜態網站。主頁將約 35 秒的六幕互動式影音嵌入中央 16:9 影片框，包含中文與英文字幕、旁白、聲音切換、略過與重播功能。

## GitHub Pages 發布

1. 將此資料夾內的所有檔案上傳到 GitHub repository 的根目錄。
2. 前往 repository 的 **Settings → Pages**。
3. 在 **Build and deployment** 選擇 **Deploy from a branch**。
4. 選擇 `main` 分支及 `/ (root)`，再按 **Save**。

網站不需要安裝套件或執行建置指令。請保留 `film`、`hero.png`、`index.html` 與 `style.css` 的相對位置。
