# Cinderella × Uber AI Landing Page

完整的 GitHub Pages 靜態網站，包含固定 Landing Page 外框、中央影片動畫、六張場景圖片、雙語字幕、MP3、自動循環播放及開始／停止按鈕。

## 上傳到 GitHub Pages

1. 解壓 ZIP，確認根目錄直接看到 `index.html`、`README.md` 與 `assets` 資料夾。
2. 將全部內容上傳到 GitHub repository 的根目錄。
3. 前往 repository 的 **Settings → Pages**。
4. 在 **Build and deployment** 選擇 **Deploy from a branch**。
5. Branch 選擇 `main`，資料夾選擇 `/(root)`，再按 **Save**。
6. 等候 GitHub Pages 顯示網站網址。

## 播放說明

- 頁面載入後，影片會在中央框內自動循環播放。
- 為符合瀏覽器自動播放規則，初次載入預設靜音；右上角按鈕可開啟聲音。
- 影片框右下角的 `❚❚`／`▶` 按鈕可停止或繼續播放。
- 所有檔案均使用相對路徑，可部署於 GitHub Pages 專案子路徑。
